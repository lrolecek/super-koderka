Staň se kóderkou

Chceš se naučit vytvářet webové stránky nebo se chceš dokonce v budoucnu uplatnit jako HTML/CSS kodérka? Tento online program je pro tebe to pravé! Nebudeme se zabývat programováním, tady budeme kódovat!

V kurzu se budeme zabývat kódováním webových stránek – převodem grafického návrhu webu do kódu, který je schopen zobrazit webový prohlížeč. Až na výjimečné jednořádkové drobnosti se nebudeme v kurzu zabývat programováním. Kurz je tedy vhodný i pro ty, co zjistily, že je baví web, ale programování jim (zatím) moc nejde.

Co budeš jako absolventka kurzu umět

  • Nakódovat komplexní webovou stránku pomocí HTML a CSS
  • Vytvářet responzivní webové stránky pro mobil, tablet nebo počítač
  • Používat pokročilé techniky pro tvorbu layoutu stránky, jako je flexbox a CSS grid
  • Psát HTML a CSS pomocí metodiky BEM (v praxi obecně asi nejpoužívanější)
  • Rozšířit možnosti CSS pomocí preprocessoru Sass
  • Vytvářet obsahově komplexní weby pomocí generátoru statických webů a napojit je na jednoduchý redakční systém
  • Psát kód stránky s ohledem na základní zásady přístupnosti
  • Pracovat s šablonovacím systémem pro snazší tvorbu komplexních webových šablon
  • …a další

Komu je kurz určen?

Všem ženám a dívkám, které se chtějí vzdělávat online v oblasti tvorby webu. Ať už se chceš do budoucna uplatnit jako kodérka nebo si vytvořit své vlastní stránky, tento kurz ti pro to poskytne všechny potřebné základy.

Je to skvělá příležitost pro všechny z různých koutů republiky i pro ty, kteří při studiu potřebují časovou flexibilitu a nemůžou být například dvakrát týdně fyzicky na daném místě ve stanoveném čase. Zda by tě kódování bavilo si vyzkoušej na našich kurzech Úvod do HTML a CSS nebo v našem online kurzu na toto téma.
